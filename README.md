## What is OpenRaw?

OpenRaw is an **open-source Agent Operating System** ‚ÄĒ not a chatbot wrapper, not a Python orchestrator. It is a full operating system for autonomous agents, built from scratch in Rust.

Traditional agent frameworks wait for you to type something. OpenRaw runs **autonomous agents that work for you** ‚ÄĒ on schedules, 24/7, building knowledge graphs, monitoring targets, generating leads, managing your social media, and reporting results to your dashboard.

The entire system compiles to a **single ~32MB binary**. One install. One command. Your agents are live.

---

## Hands ‚ÄĒ Autonomous Capability Packages

**Hands** are OpenRaw's core innovation ‚ÄĒ pre-built autonomous modules that run independently, on schedules, without prompting. This is not a chatbot. This is an agent that wakes up at 6 AM, researches your competitors, builds a knowledge graph, scores the findings, and delivers a report before you've had coffee.

Each Hand bundles a system prompt, skill reference, tool manifest, and guardrails ‚ÄĒ compiled directly into the binary. No downloading, no pip install, no Docker pull.

| Hand | What It Does |
|------|-------------|
| **Clip** | YouTube ‚Üí vertical shorts with captions and thumbnails. 8-phase pipeline. FFmpeg + yt-dlp + 5 STT backends. |
| **Lead** | Daily prospect discovery. Enriches, scores 0‚Äď100, deduplicates, delivers qualified leads in CSV/JSON/Markdown. |
| **Collector** | OSINT-grade intelligence. Monitors continuously ‚ÄĒ change detection, sentiment tracking, knowledge graph construction. |
| **Predictor** | Superforecasting engine. Calibrated reasoning chains, confidence intervals, Brier score accuracy tracking. |
| **Researcher** | Deep autonomous researcher. CRAAP-criteria source evaluation, cited reports, multi-language support. |
| **Twitter** | Autonomous X account manager. 7 rotating content formats, approval queue ‚ÄĒ nothing posts without your OK. |
| **Browser** | Web automation. Navigates sites, fills forms, handles workflows. Mandatory purchase approval gate. |

openraw-channels    40 messaging adapters with rate limiting, DM/group policies
openraw-memory      SQLite persistence, vector embeddings, canonical sessions, compaction
openraw-types       Core types, taint tracking, Ed25519 manifest signing, model catalog
openraw-skills      60 bundled skills, SKILL.md parser, Hub marketplace
openraw-hands       7 autonomous Hands, HAND.toml parser, lifecycle management
openraw-extensions  25 MCP templates, AES-256-GCM credential vault, OAuth2 PKCE
openraw-wire        OFP P2P protocol with HMAC-SHA256 mutual authentication
openraw-cli         CLI with daemon management, TUI dashboard, MCP server mode
openraw-desktop     Tauri 2.0 native app (system tray, notifications, global shortcuts)
openraw-migrate     OpenClaw, LangChain, AutoGPT migration engine

## 16 Security Layers

| # | System | What It Does |
|---|--------|-------------|
| 1 | **WASM Dual-Metered Sandbox** | Tool code runs in WebAssembly with fuel metering + epoch interruption. A watchdog thread kills runaway code. |
| 2 | **Merkle Hash-Chain Audit Trail** | Every action is cryptographically linked to the previous one. Tamper with one entry and the chain breaks. |
| 3 | **Information Flow Taint Tracking** | Labels propagate through execution ‚ÄĒ secrets tracked from source to sink. |
| 4 | **Ed25519 Signed Agent Manifests** | Every agent identity and capability set is cryptographically signed. |
| 5 | **SSRF Protection** | Blocks private IPs, cloud metadata endpoints, and DNS rebinding attacks. |

## Build from Source (GitHub)

When you clone the repo, you get everything ‚ÄĒ CLI, Desktop app, and all crates. One build produces both binaries:

```bash
git clone https://github.com/your-username/openraw
cd openraw
cargo build --release
```

**Output:**
- `target/release/openraw.exe` ‚ÄĒ CLI (TUI, commands, daemon)
- `target/release/openraw-desktop.exe` ‚ÄĒ Native desktop app (WebView2 window)

**First run:**
```bash
.\target\release\openraw.exe init    # configure API keys
.\target\release\openraw.exe start  # daemon, or run openraw-desktop.exe
# Dashboard ‚Üí http://localhost:4200
```

**Requirements:** Rust, Visual Studio Build Tools (Windows), WebView2 for desktop app.
